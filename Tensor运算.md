# 张量的基本概念
 张量是一个多维数组。更正式地说，一个 N 阶张量是 N 个向量空间元素的张量积，每个向量空间都有自己的坐标系。  
 张量的阶数（the order of a tensor）也称为维数（dimensions）、模态（modes）、或方式（ways）。  
 一阶张量是一个矢量，二阶张量是一个矩阵，三阶或更高阶的张量叫做高阶张量。
